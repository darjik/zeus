Order Guidelines
Order Creation Tips:
  • Select the customer first to enable contact person selection
  • Set expected delivery date for better planning
  • Include customer's PO number for reference
  • Provide detailed requirements for production team
Status Information:
  • Pending: Order received, awaiting confirmation
  • Confirmed: Order confirmed and accepted
  • In Development: Product development in progress
  • Testing: Quality testing phase
  • Ready for Dispatch: Ready to ship
  • Dispatched: Order shipped
  • Delivered: Order delivered to customer
